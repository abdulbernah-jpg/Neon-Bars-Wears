# Neon Bars Wear

A simple static landing page for Neon Bars Wear, a streetwear brand inspired by music, creativity, and self-expression.

## Files

- `index.html` — landing page structure and brand messaging
- `collection.html` — product collection page with mockup visuals
- `contact.html` — contact page with social and email details
- `styles.css` — styling with neon accents, dark theme, and responsive layout
- `.github/workflows/pages.yml` — GitHub Actions workflow to publish the site

## Usage

1. Open `index.html` in a browser for a local preview.
2. Edit the contact details and replace placeholder mockup sections with real product images.
3. Push the repository to GitHub to enable deployment.

## Deployment

Once the repository is pushed, the site should be accessible at:

`https://abdulbernah-jpg.github.io/Neon-Bars-Wears/`

If the site is not yet live after pushing, the workflow file in `.github/workflows/pages.yml` will publish the project to GitHub Pages on the `gh-pages` branch.
## Branding

- Color palette: Black, White, Neon Blue, Neon Purple, Dark Gray
- Headings: Bebas Neue
- Body text: Montserrat (with a sans-serif fallback)
